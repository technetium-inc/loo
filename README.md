# loo
The loo programming language, created by technetium Inc

A simple hello world program
```bf
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
++#+++++++++++++++++++++++++++++#+++++++##+++#------------------------
-------------------------------------------------------#++++++++++++++
+++++++++++++++++++++++++++++++++++++++++#++++++++++++++++++++++++#+++#
------#--------#------------------------------------------------------
--------------#+#
```

## `+`
Increments the ascii value

## `-`
Decrements the ascii value

## `#`
Print the current character

## `;`
Stop the execution
